    页面定制化比较严重, 不够轻量. 底层基于 tiptap(prosemirror)
3. [tiptap-markdown](https://github.com/aguingand/tiptap-markdown)
    也有问题
4. [md-editor-v3](https://github.com/imzbf/md-editor-v3)
    vue版本, 底层基于 codemirror 实现
5. [milkdown](https://github.com/Milkdown/milkdown) 
    底层基于 prosemirror + remark 实现. markdown 语法中 code block 基于 codemirror 实现
