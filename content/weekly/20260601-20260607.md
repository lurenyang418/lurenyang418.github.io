## ssh 警告处理

```shell
ssh -o "UserKnownHostsFile=/dev/null" -o "StrictHostKeyChecking=no" -i ~/.ssh/id_rsa 
```

等同于在 `~/.ssh/config` 配置

```ini
Host *
  StrictHostKeyChecking no
  UserKnownHostsFile=/dev/null
```

## swap 分区关闭
```shell
# 临时关闭
sudo swapoff -a
# 永久关闭（注释 /etc/fstab 中的 Swap 行）
sudo sed -i '/swap/s/^/#/' /etc/fstab
# 验证
free -h  # Swap 行应全为 0
```

## linux 清除缓存并释放内存
```shell
# 0：不释放（系统默认值）
# 1：释放页缓存
# 2：释放dentries和inodes
# 3：释放所有缓存
sync && echo 3 >> /proc/sys/vm/drop_caches
```

## 使用 lego 申请腾讯云 https
测试环境地址: https://acme-staging-v02.api.letsencrypt.org/directory
正式环境: https://acme-v02.api.letsencrypt.org/directory
使用的 DNS 方式，支持泛域名`*.example.com`
[DNSPOD 文档](https://go-acme.github.io/lego/dns/dnspod/index.html)
```bash
# tencent old -> dnspod
export DNSPOD_API_KEY="<DP_Id>,<DP_Key>"
# tencent new -> tencentcloud
export TENCENTCLOUD_SECRET_ID=<your-id>
export TENCENTCLOUD_SECRET_KEy=<your-key>
lego --email="<email>" --dns dnspod --domains="<example.com>" --domains="*.example.com" run
```

## docker 找到一个镜像的子镜像
```bash
docker inspect --format='{{.Id}} {{.Parent}}' $(docker images --filter since=<image_id>-q) | grep <image_id> | awk '{print $1}' | cut -d: -f2
```

## openssl 生成32位随机密码
